在 Kether 中，我们通常使用 [b]*[/b] 来定义数据
例如 *true、*0、*1、*12138、*minecraft、*"Hello World!" 等等...

你可能会认为 *true 是布尔型中的 true 而 *0、*1、*12138 是整数型数据，分别代表了数字0，1和12138

但仔细研究会发现，这样定义的数据在 Kether 里都表示为 String 字符串型数据
